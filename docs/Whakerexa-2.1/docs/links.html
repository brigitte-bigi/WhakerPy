Link Controller

by Whakerexa

Description

Manage user interactions on clickable elements that open URLs.

What it is for

LinkController standardizes how clickable elements open URLs. It makes both mouse click and Enter behave the same way, and lets you target a new tab, the current page, or an iframe preview.

Rules

  • Activation: click or Enter.
  • URL: href is used first; if missing/empty, data-href is used.
  • Target (data-target): _blank (default), _self, or an iframe id.
  • If data-target matches an existing <iframe> id, the URL is loaded into that iframe.
  • Default navigation is prevented (so behavior stays consistent).
