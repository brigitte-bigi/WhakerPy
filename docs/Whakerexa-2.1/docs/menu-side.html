Description

This page demonstrates a “side navigation” layout (desktop-style). The HTML of this page is the reference (doc + demo).

What this example shows

  • A top navigation bar: #nav-content
  • A side menu container: aside.appmenu with submenus
  • Menu controls: #menu-button (mobile toggle) and #pin-menu (pin)

CSS provides the layout (menu.css). JavaScript is only used to open/close menus and manage submenus (MenuManager).
